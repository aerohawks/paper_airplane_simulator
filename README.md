# 종이비행기 시뮬레이터 🛩️

웹 기반 대화형 종이비행기 물리 시뮬레이터입니다. 실제 공기역학 원리를 적용하여 종이비행기의 비행을 시뮬레이션합니다.

## 🎮 주요 기능

### 비행기 설계
- **날개 크기**: 비행기의 양력과 공기저항에 영향
- **무게**: 비행기의 낙하 속도와 안정성 조절
- **동체 길이**: 비행기의 무게중심과 안정성 결정
- **꼬리 크기**: 방향 안정성과 회전 특성 영향

### 발사 설정
- **발사력**: 비행기의 초기 속도 (5~50)
- **발사 각도**: 상승각 (-30° ~ 60°)
- **바람 강도**: 시뮬레이션 중 바람의 영향 (0~15)

### 환경 설정
- **중력**: 행성의 중력 크기 (5~15 m/s²)
- **공기 저항**: 드래그 계수 (0.01~0.5)

### 실시간 통계
- 비행 시간
- 최대 높이
- 최대 거리
- 최대 속도

## 🔬 물리 시뮬레이션

### 구현된 물리 모델

1. **중력**: 선택된 중력 계수에 따라 연속 적용
2. **공기 저항 (드래그)**:
   ```
   F_drag = -0.5 * ρ * v² * Cd * A * v_direction
   ρ = 1.225 kg/m³ (공기 밀도)
   Cd = 드래그 계수
   A = 날개 면적
   ```

3. **양력**:
   ```
   F_lift = 0.5 * ρ * v² * Cl * A
   Cl = 양력 계수 (대략 0.5)
   ```

4. **바람**: 벡터 형태의 외부 힘 적용

### 사용된 라이브러리

- **Three.js**: 3D 그래픽 렌더링
- **Cannon-es**: 물리 엔진 (강체 동역학)

## 🛠️ 개발

### 코드 구조

- **Simulator**: 전체 시뮬레이션 관리 (Three.js, Cannon-es)
- **PaperAirplane**: 비행기 3D 모델 및 물리 바디
